## NLW Copa - Mobile
<br>
Uma aplicação mobile incrível da Rocketseat utilizando os serviços OAuth para realizar o login com a conta do Google, retornando informações pessoais do usuário dentro do aplicativo. 
<br>
<br>

## Suas Funcionalidades
<br>
<p>
 ☑ Login integrado com o OAuth Google Console
</p>
<p>
 ☑ Criar novo bolão
</p>
<p>
 ☑ Participar de um novo bolão existente
</p>
<p>
 ☑ Dar palpites sobre os próximos jogos
</p>
<p>
 ☑ Ranking competitivo entre os participantes dos bolões
</p>
